# Turkic Flags Example Application 🇹🇷 🇦🇿 🇰🇿 🇰🇬 🇺🇿 🇹🇲 🇨🇾

This application demonstrates the interactive features, widgets, and vector graphics provided by the [`turkic_flags`](https://pub.dev/packages/turkic_flags) package.

## 🌟 What's Included

- **Interactive Catalog Grid**: Displays all 41 Turkic world flags (including the 16 Great Turkic Empires) in responsive, symmetrical cards (`TurkicFlagCard`) using uniform aspect ratios (`flagAspectRatio: 1.5`).
- **Dynamic Search & Filtering**: Filter flags by category (Independent States, Autonomous Republics, Historical Empires) or search by name in real-time.
- **Dual-Language & Theme Controls**: Switch between Turkish and English descriptions, as well as Light and Dark theme modes.
- **High-Resolution Detail View**: Inspect authentic SVG graphics at large scale, view primary color hex codes, and copy Flutter implementation code snippets with a single tap.

## 🚀 Running the Example

Make sure you have Flutter installed, then run:

```bash
flutter pub get
flutter run
```
